## Image Guidelines

1. **Quality**: Use high-resolution images (minimum 1200px width for landscape photos)
2. **Format**: JPG for photos, PNG for logos/graphics
3. **Optimization**: Compress images to reduce file size while maintaining quality
4. **Naming**: Use lowercase with underscores (e.g., `lga_chairman.jpg`)
5. **Copyright**: Ensure you have permission to use all images

## How to Add Images

1. Place image files in this `assets/` folder
2. Update the HTML file to reference images:
   ```html
   <img src="assets/image_name.jpg" alt="Description" />
   ```
